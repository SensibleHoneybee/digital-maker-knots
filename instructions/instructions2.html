Step two: Put the "skeleton" HTML into the page

Copy this code into the file knots.html which is open in Visual Studio Code:

<!DOCTYPE html>
<html>
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://www.w3schools.com/w3css/3/w3.css">
  <head>
    <title>Page Title</title>
  </head>
  <body>

    <!-- The navigation bar goes here -->

    <!-- This is the main body of the webpage -->
    <div id="home" class="w3-content" style="margin-top: 50px;">
      <h1>My First Heading</h1>
      <p>My first paragraph.</p>
    </div>

  </body>
</html>

The parts of the code which define the content is contained in elements:

What is an HTML Element?

An HTML element is defined by a start tag, some content, and an end tag:

<tagname> Content goes here... </tagname>

The HTML element is everything from the start tag to the end tag:

<h1>My First Heading</h1>
   (h1 means a level 1 heading)

<p>My first paragraph.</p>
   (p means a paragraph of text)

Note: Some HTML elements have no content (like the <br> element). These elements are called empty elements. Empty elements do not have an end tag!
