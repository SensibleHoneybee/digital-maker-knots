Step three: Run the web page

  1. In Visual Studio Code, with your HTML file open, click "Ctrl + O", or go to the File menu and select "Open File..."
  2. In the Open File dialog, you should see your index.html file listed.
  3. Right click on the index.html file.
  4. From the drop down menu, select Open (not "Select")
  5. Your file should open in the computer's browser
